## 命名规范

### 文件命名
```
主题_描述_v版本号.扩展名
```

示例：
- `抽纸横向对比评测_25款_完整版_v5.xlsx`
- `朋友圈文案_3月26日_20点_20260326.md`
- `小龙虾赋能营_财务推演_v1.0.md`

### 文件夹命名
```
序号_分类名称
```

示例：
- `01_Projects`
- `02_抽纸对比`
- `03_小龙虾赋能营`

---

## 使用规则

1. **新项目** → 放入 `01_Projects/`
2. **项目完成** → 移动到 `04_Archive/`
3. **临时文件** → 放入 `05_Inbox/`，定期清理
4. **产出物** → 按类型放入 `output/` 对应文件夹
5. **每日记忆** → 自动写入 `memory/YYYY-MM-DD.md`
