## Memory Skill 列表（20:09）

### 已安装的 Memory Skill（2个）

| Skill | 描述 | 用途 |
|-------|------|------|
| **memory-hygiene** | 审计、清理和优化向量记忆（LanceDB） | 记忆膨胀、token浪费、自动维护 |
| **memory-setup** | 启用和配置记忆搜索持久化上下文 | 解决"金鱼脑"、设置记忆系统 |

### 详细说明

**1. memory-hygiene**
- **功能：** 保持向量记忆精简，防止垃圾记忆浪费token
- **核心命令：**
  - `memory_recall query="*" limit=50` - 审计记忆内容
  - `rm -rf ~/.clawdbot/memory/lancedb/` - 清空记忆
  - `memory_store text="<fact>"` - 重新植入关键事实
- **使用场景：** 记忆膨胀、无关自动召回、设置月度维护

**2. memory-setup**
- **功能：** 将agent从金鱼变成大象，配置持久记忆
- **核心功能：**
  - 启用记忆搜索配置
  - 创建记忆目录结构
  - 初始化MEMORY.md
  - 设置向量搜索
- **使用场景：** 初次设置记忆、修复"金鱼脑"、配置记忆搜索

### 推荐安装命令
```bash
# 基础记忆系统（必选）
openclaw skill install memory-setup

# 记忆维护（推荐）
openclaw skill install memory-hygiene
```

### 使用顺序
先安装 `memory-setup` 建立系统，再用 `memory-hygiene` 定期维护。
