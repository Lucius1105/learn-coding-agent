# 2026-03-17 工作记录

## 朋友圈文案学习 - 老徐版本 vs 我的版本

### 老徐写的第2条（南宁专场）：

```
因为一句话，兑现一个承诺。

昨天收到一条消息。
南宁一个做餐饮的老板说：能不能来南宁做一场？

我盯着屏幕愣了一下。
然后回：好，安排！

就3️⃣个名额，
想变得更好的老板可以预约。
「你若信任所托，我必全力以赴。」
```

### 我的版本（问题分析）：

| 维度 | 老徐版本 | 我的版本 | 差距 |
|------|---------|---------|------|
| **开头** | "因为一句话，兑现一个承诺" - 金句前置，有态度 | "昨天收到一条消息" - 平铺直叙 | 老徐开头有观点，我是场景 |
| **节奏** | 短句+emoji（3️⃣），视觉有停顿 | 纯文字，节奏单一 | 老徐用emoji做视觉锚点 |
| **金句** | 「你若信任所托，我必全力以赴」- 书名号强调 | "你若相信，我必全力以赴" - 普通引号 | 老徐用「」突出，且用词更精准（"信任所托" vs "相信"） |
| **结尾** | "想变得更好的老板可以预约" - 正向引导 | "有问题的老板可以预约" - 问题导向 | 老徐用正向语言，我用痛点语言 |
| **态度** | "好，安排！" - 干脆利落 | "好，下周。就3个名额" - 解释太多 | 老徐更果断，我解释过度 |

### 学到的教训：

1. **开头要有态度，不只是场景**
   - 老徐："因为一句话，兑现一个承诺"（观点先行）
   - 我："昨天收到一条消息"（纯场景）

2. **用emoji做视觉停顿**
   - 3️⃣ 比 "3个名额" 更有记忆点
   - 数字emoji是朋友圈的视觉锚点

3. **金句要突出，用「」书名号**
   - 「你若信任所托，我必全力以赴」
   - 不是"你若相信"，是"你若信任所托"（更精准）

4. **正向引导 > 问题导向**
   - 老徐："想变得更好的老板"
   - 我："有问题的老板"
   - 正向语言更有能量

5. **果断 > 解释**
   - 老徐："好，安排！"
   - 我："好，下周。就3个名额"
   - 不要解释太多，信任老徐的判断

6. **个人态度要前置**
   - 老徐第一句就是观点："因为一句话，兑现一个承诺"
   - 我第一句只是叙事，没有态度

### 核心差距：

**老徐的朋友圈 = 观点先行 + 场景佐证 + 金句收尾**
**我的朋友圈 = 场景叙事 + 解释说明 + 口号结尾**

老徐是**有观点的人**，我是**记录事件的人**。
