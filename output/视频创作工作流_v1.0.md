## 工作流总览

```
┌─────────────────────────────────────────────────────────────────┐
│  Phase 1: 灵感捕获（输入）                                        │
├─────────────────────────────────────────────────────────────────┤
│  触发方式：                                                      │
│  - 老徐口述录音（10-15分钟）                                       │
│  - 日常观察记录（案例/对话/阅读）                                   │
│  - 热点事件关联（可选，非必须）                                     │
│                                                                  │
│  输出物：灵感卡片                                                  │
│  - 主题一句话                                                      │
│  - 核心观点（1-3个）                                               │
│  - 案例素材（真实来源）                                            │
│  - 目标受众                                                        │
│  - 预期情绪（温暖/犀利/启发）                                       │
│                                                                  │
│  存储位置：~/obsidian-mirror/.../02_流量获取与市场矩阵/📂 00_素材库与灵感库/灵感闪现/ │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Phase 2: 脚本撰写（/video）                                      │
├─────────────────────────────────────────────────────────────────┤
│  负责人：video-official 技能（A1989）                              │
│                                                                  │
│  输入：灵感卡片 或 老徐口述主题                                    │
│                                                                  │
│  输出物：完整脚本文档                                              │
│  - 脚本正文（分段时间轴）                                          │
│  - 分镜与B-roll指导                                                │
│  - BGM配置建议                                                     │
│  - 标题与封面建议                                                  │
│  - 真善美自检                                                      │
│  - 内容资产价值评估                                                │
│                                                                  │
│  质量标准：                                                        │
│  - 真善美三关（真实/善意/美感）                                    │
│  - 内容资产三标准（价值共鸣/理念认同/穿越周期）                     │
│                                                                  │
│  存储位置：~/.openclaw/workspace/output/                          │
│  命名规范：video_主题_日期_版本.md                                  │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Phase 3: 剪辑指导（/edit）                                       │
├─────────────────────────────────────────────────────────────────┤
│  负责人：video-editor 技能（A1989）                                │
│                                                                  │
│  输入：脚本文档                                                    │
│                                                                  │
│  输出物：剪辑方案文档                                              │
│  - 分镜表（时间/画面/B-roll/剪辑/字幕/BGM）                        │
│  - B-roll素材清单（含来源建议）                                     │
│  - BGM详细配置（曲目/音量时间轴）                                   │
│  - 字幕设计规范                                                    │
│  - 封面设计方案                                                    │
│  - 导出检查清单                                                    │
│                                                                  │
│  执行者：老徐自剪 / 外包剪辑师                                     │
│                                                                  │
│  工具：剪映专业版（推荐）                                          │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Phase 4: 拍摄执行（人工）                                        │
├─────────────────────────────────────────────────────────────────┤
│  负责人：老徐 / 摄影师                                             │
│                                                                  │
│  拍摄清单（按脚本）：                                              │
│  - [ ] 主画面：口播视频（各段落全覆盖）                             │
│  - [ ] B-roll：脚本中标记的所有补充画面                             │
│  - [ ] 封面素材：专用拍摄画面                                       │
│                                                                  │
│  拍摄规范：                                                        │
│  - 机位稳定（三脚架/云台）                                         │
│  - 光线充足（面部明亮，无阴影）                                     │
│  - 背景简洁（不杂乱，符合品牌调性）                                  │
│  - 收音清晰（可用领夹麦）                                          │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Phase 5: 剪辑制作（人工+工具）                                    │
├─────────────────────────────────────────────────────────────────┤
│  负责人：老徐自剪 / 剪辑师                                         │
│                                                                  │
│  按剪辑方案执行：                                                  │
│  1. 导入素材（口播+B-roll）                                        │
│  2. 粗剪：按时间轴拼接主画面                                        │
│  3. 精剪：插入B-roll，调整节奏                                      │
│  4. 加字幕：逐句校对，关键词高亮                                     │
│  5. 配BGM：按音量时间轴调整                                         │
│  6. 做封面：按设计方案制作                                          │
│  7. 导出：1080×1920，30fps，MP4                                     │
│                                                                  │
│  必过检查清单（见 video-editor skill）                             │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│  Phase 6: 发布与追踪（/publish，待建）                             │
├─────────────────────────────────────────────────────────────────┤
│  负责人：video-publish 技能（待建）                                │
│                                                                  │
│  发布动作：                                                        │
│  - 视频号发布（主平台）                                            │
│  - 小红书同步（图文版）                                            │
│  - 朋友圈文案（3条）                                               │
│                                                                  │
│  数据追踪：                                                        │
│  - 24小时：播放量/点赞/评论                                        │
│  - 7天：收藏/转发/私域导流                                         │
│  - 30天：长尾数据/搜索来源                                         │
│                                                                  │
│  复盘记录：填入「内容发布数据复盘表」                                │
└─────────────────────────────────────────────────────────────────┘
```

## 内容资产标准

每个产出必须通过三项检验：

### 1. 价值共鸣
- 目标受众看完会觉得「这说的是我」
- 解决真实痛点，不是制造焦虑
- 有具体场景，不是抽象道理

### 2. 理念认同
- 传递雨后甘霖的核心价值观
- 「借商修人，以商度人」
- 方法可落地，不是空谈

### 3. 穿越周期
- 不追短期热点
- 一年后看依然有收获
- 建立长期信任，不是一次性流量
