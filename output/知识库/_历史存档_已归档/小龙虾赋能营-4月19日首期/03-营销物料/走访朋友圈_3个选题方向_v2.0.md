# 走访朋友圈 · 3个选题方向

> 结合真实案例，展示服务能力

---

## 方向A：陪老板变强（服务价值型）

**核心信息：** 我们不只是教AI，是帮老板找回时间、平衡生活

**文案：**
```
今天走访了一位从外地回柳州创业的朋友。

当初决定回来，他说就是想多陪家人。但真干起来才发现——每天被琐事困住，陪孩子的时间反而少了。

聊了一下午，我们一起理了三个问题：
1. 哪些事可以交给AI？
2. 哪些事必须亲自做？
3. 省下来的时间，怎么花在刀刃上？

临走时他说："终于有人懂我了，不是催我拼命干，是帮我找方法。"

这就是我们做「雨后甘霖」的初衷——
陪老板变强，但不以牺牲生活为代价。

如果你也在找这个平衡点，欢迎找我聊聊。
```

**适合：** 强调服务理念，吸引同频客户

---

## 方向B：诊断现场（专业展示型）

**核心信息：** 展示咨询过程，体现专业能力

**文案：**
```
今天下午在一位创业者办公室坐了3小时。

没讲大道理，就做了三件事：
✓ 看他怎么过一天（时间花在哪）
✓ 算他哪些活最耗时（重复性工作占比）
✓ 找出能马上优化的点（AI替代方案）

发现他每天4小时在做"机械劳动"——整理数据、回复常见问题、盯流程进度。

而这些，AI可以帮他省出70%的时间。

不是他不够努力，是努力的方向需要调整。

雨后甘霖的商业诊断，就是帮你找到这个"调整点"。
```

**适合：** 展示具体服务内容，建立专业信任

---

## 方向C：创业者共鸣（情感连接型）

**核心信息：** 引发同类创业者共鸣，建立情感链接

**文案：**
```
今天拜访的这位老板，说了一句话让我印象深刻：

"在外面打拼十年，以为回来创业能轻松点陪家人。结果发现，创业比打工还忙。"

这话我听了太多遍。

很多老板不是不想慢下来，是不知道怎么慢下来——
事情推着你走，停不下来。

我们的价值，就是帮你找到那个"停得下来的方法"：
让该自动化的自动化，让你专注真正重要的事。

如果你也有同感，评论区聊聊？
```

**适合：** 引发互动评论，扩大传播
