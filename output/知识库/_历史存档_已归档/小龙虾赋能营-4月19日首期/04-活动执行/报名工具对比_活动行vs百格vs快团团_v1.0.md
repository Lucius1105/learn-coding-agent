## 二、详细分析

### 1. 活动行（EventBank）

**费率结构：**
```
用户付款：¥498
平台扣3%：¥498 × 3% = ¥14.94
实际到账：¥483.06
提现再扣0.6%：¥483.06 × 0.6% = ¥2.90
最终到手：¥480.16

实际费率：3.6%
30人总收入：¥14,940
实际到手：¥14,404（损失¥536）
```

**退款政策：**
- 用户申请退款 → 全额退给用户
- 已扣的3%手续费**不退还**
- 例：退款¥498，你损失¥14.94

**优点：**
- 品牌知名度高，用户信任
- 活动模板丰富
- 支持现场签到扫码

**缺点：**
- 费率相对较高
- 有竞品活动推荐（轻微导流）

---

### 2. 百格活动（推荐）

**费率结构：**
```
基础版：免费创建，2%手续费
标准版：¥199/月，1.5%手续费

按基础版计算：
用户付款：¥498
平台扣2%：¥498 × 2% = ¥9.96
实际到账：¥488.04
提现再扣0.6%：¥488.04 × 0.6% = ¥2.93
最终到手：¥485.11

实际费率：2.6%
30人总收入：¥14,940
实际到手：¥14,553（损失¥387）
```

**退款政策：**
- 同活动行，手续费不退
- 但基数小（2% vs 3%），损失更少

**优点：**
- B端培训专用，界面商务
- 广告最少，体验干净
- 表单字段最灵活
- 数据导出格式规范

**缺点：**
- 知名度不如活动行
- 用户需要适应新平台

---

### 3. 快团团

**费率结构：**
```
用户付款：¥498
平台不扣手续费
提现扣0.6%：¥498 × 0.6% = ¥2.99
最终到手：¥495.01

实际费率：0.6%
30人总收入：¥14,940
实际到手：¥14,850（仅损失¥90）
```

**退款政策：**
- 全额退款，无任何手续费损失
- 这是最大优势

**优点：**
- 费率最低
- 大家熟悉拼多多生态
- 支持"拼团"玩法（自带传播）
- 资金实时到账

**缺点：**
- 看起来像卖货，不够正式
- 信息收集能力弱
- 不适合B端培训场景

## 四、我的最终建议

### 首选：百格活动（平衡型）

**理由：**
1. 费率适中（2.6%），比活动行省¥150
2. B端培训专用，客户体验好
3. 广告少，显得专业
4. 数据导出方便，客服跟进容易

**操作流程：**
```
1. 注册百格活动账号
2. 创建「小龙虾AI赋能营」
3. 设置时间：2026年4月19日 14:00-18:00
4. 设置地点：柳州XX酒店（待定）
5. 设置票种：
   - 单人票 ¥498（限量20张）
   - 双人票 ¥798（限量10组）
6. 开启微信支付
7. 设置必填字段：
   - 姓名
   - 手机号
   - 公司名称
   - 职位
   - 最想解决的痛点（选填）
8. 发布并生成海报
```

---

### 备选：拉卡拉+腾讯问卷（零费率）

**如果你特别在意手续费：**

```
工具组合：
├── 腾讯问卷：收集报名信息（免费）
├── 拉卡拉收款码：用户扫码付款
└── 人工核对：客服确认后标记"已付款"

用户路径：
填问卷 → 看到收款码 → 截图保存 → 扫码付款 → 
回传付款截图 → 客服确认 → 报名成功

成本：¥0手续费
缺点：操作繁琐，流失率高（预计多流失20%）
```

**只建议在以下情况使用：**
- 报名人数很少（<10人）
- 全是熟人介绍，信任度高
- 愿意承担人工核对成本
