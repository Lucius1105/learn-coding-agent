# 【采购决策报告】无线麦克风_v2.2（最终决策版）

>

---

## 1️⃣ 核心痛点

| 设备 | 问题 |
|------|------|
| MacBook Air M4 | 内置麦克风时好时坏，语音输入不稳定 |
| Bose Mini 2 | 连接后麦克风识别不了 |
| 有线耳麦 | 麦克风能用，但声音从耳机出，线短不舒服 |
| 飞度 FD40 | 要接3.5转Lightning线，不优雅，没收纳盒，天线乱 |

### 理想状态
- **vibe coding**：Mac语音输入稳定，效率提升（50字/分 → 200字/分）
- **短视频**：拿出即用，无需接线，收纳方便，降低拍摄阻力

---

## 2️⃣ 解决方案

### 无线麦克风全家桶版

**Mac vibe coding：**
- 接收器USB-C直插Mac，发射器夹衣领
- 声音继续从Bose外放，麦克风无线输入

**iPhone短视频：**
- 接收器Lightning直插iPhone，发射器夹衣领
- 拿出即用，拍完放回充电盒

## 4️⃣ 验机检查清单

- [ ] 电池续航正常（充满能用4-6小时）
- [ ] 三个接收器都能正常工作
- [ ] 发射器无划痕、无摔痕
- [ ] 充电盒开合正常
- [ ] 配件齐全（发射器×2、接收器×3、充电盒、线材）
- [ ] 卖家信誉良好，有实拍图
- [ ] 购买时间2024年后，使用<1年

### 电池验机方法

| 方法 | 操作 | 判断标准 |
|------|------|----------|
| 续航测试 | 问卖家：充满电能用多久？ | 官方6-8小时，实际4-6小时正常 |
| 充电次数 | 问购买时间、使用频率 | 2024年后购买，使用<1年为佳 |
| 外观检查 | 看充电触点是否氧化 | 氧化=使用久/保养差 |
| 功能测试 | 要求发视频：开机→连接→录音 | 连接稳定，无断连 |
| 保修期 | 问是否还在保修期 | 有保修=保险 |
