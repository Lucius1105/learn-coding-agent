## 使用场景

### 场景1：口播视频（固定机位）

```
设备：iPhone 13 Pro + 猛犸M2
设置：
  - 手机固定在三脚架/支架上
  - 猛犸M2发射器磁吸领口
  - 接收器Lightning直插手机
  - 拿出即用，拍完放回充电盒
```

### 场景2：vibe coding（Mac语音输入）

```
设备：MacBook Air M4 + 猛犸M2
设置：
  - 接收器USB-C直插Mac
  - 发射器夹衣领
  - 声音从Bose Mini 2外放
  - 麦克风无线输入，稳定识别
```

### 场景3：短视频拍摄（移动场景）

```
设备：iPhone 13 Pro + 猛犸M2
设置：
  - 手持或简易支架
  - 猛犸M2隐形佩戴
  - 无线收音，自由移动
```
