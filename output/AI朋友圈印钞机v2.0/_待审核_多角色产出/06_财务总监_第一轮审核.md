**预案A：退款率过高（>10%）**

```
触发条件：退款率>10% 或 连续3单退款

财务影响：
- 直接损失：退款金额
- 间接损失：获客成本沉没

应对措施：
1. 立即停止销售，复盘退款原因
2. 已交付客户主动回访，防止连锁退款
3. 调整销售话术，降低过度承诺
4. 前置客户筛选，提高门槛

恢复条件：
- 连续5单无退款
- 客户NPS>8分
```

**预案B：投放ROI不达标（<2:1）**

```
触发条件：连续2周ROI<2:1 或 单客获客成本>¥1,000

财务影响：
- 投放资金亏损
- 获客成本过高，侵蚀利润

应对措施：
1. 立即停止投放
2. 复盘素材和定向设置
3. 回归私域转化，暂停付费流量
4. 优化素材后小预算测试

恢复条件：
- 测试ROI>3:1
- 单客获客成本<¥500
```

**预案C：现金流断裂**

```
触发条件：账上现金<¥30,000 或 连续2周无法支付成本

财务影响：
- 无法支付老徐时间成本
- 无法支付助理工资
- 项目停摆

应对措施：
1. 立即停止所有支出（除必要成本）
2. 暂停新单销售，消化存量订单
3. 紧急催收应收账款
4. 考虑降价促销快速回款

- 账上现金>¥50,000
```
