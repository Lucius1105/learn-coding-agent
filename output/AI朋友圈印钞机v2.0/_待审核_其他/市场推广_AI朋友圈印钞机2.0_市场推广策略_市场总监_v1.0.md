## 二、内容营销策略

### 1. 视频号内容规划

- **账号名**：阿徐商业教练（保持现有）

- **视觉风格**：真人出镜为主，背景简洁（办公室/咖啡馆），突出专业感

#### 内容类型与比例

| 类型 | 占比 | 目的 | 示例 |

- **预热期**：每天1条（持续7天）
- **正式发售期**：每天2条（早8点+晚8点）
- **常态运营**：每周3-5条

#### 选题库（20个）

1. 这个律师朋友圈改了3条，当月接了2个案子

7. 财务顾问用朋友圈，一年多做200万业绩
8. 咨询师不发专业术语，客户反而更信任

**干货方法类（7个）：**

### 3. 小红书内容规划

#### 账号定位

- **账号名**：阿徐商业教练（或新号"朋友圈印钞机"）
- **简介**：帮高客单价专业人士用朋友圈自动获客｜连续创业13年｜已服务60+企业主
- **视觉风格**：统一封面模板（大字标题+人物半身），色调专业稳重（深蓝/金色）

#### 爆款笔记结构

**结构模板：**
```
【封面】大字标题：痛点+数字+结果
  例："律师朋友圈3不改，客户永远找不到你"

【标题】关键词前置+情绪词
  例："高客单价业务必看｜朋友圈变现的3个隐形开关"

【正文】
  开头：钩子（痛点共鸣/反常识观点）
  中间：3-5个要点，每点配解释
  结尾：引导互动（"你遇到过这个问题吗？评论区聊聊"）
  标签：#朋友圈营销 #高客单价 #私域变现 #律师获客 #咨询师
```

**爆款公式：**
- 痛点词 + 数字 + 解决方案
- 身份标签 + 反常识观点 + 方法论
- 案例故事 + 数据结果 + 可复制方法

#### 引流路径设计

```
小红书笔记 → 评论区引导私信 → 自动回复（飞书/公众号） → 添加微信 → 朋友圈培育 → 产品转化
```

**评论区话术：**
- "详细方法整理了一份文档，私信【朋友圈】发你"
- "这个问题我们专门做了个7天训练营，需要的话私信"

## 四、会销方案（可选）

**主题：** "AI时代的朋友圈印钞术——高客单价业务的私域变现闭门会"

**形式：** 线上直播（腾讯会议/视频号直播）+ 线下小型沙龙（可选）

**时长：** 2小时（线上）/ 半天（线下）

**人数：** 线上不限/线下20-30人

> X总好，我是阿徐。4月X号我准备了一场关于"AI朋友圈变现"的闭门分享，专门聊高客单价业务怎么用朋友圈自动获客。现场会拆解3个真实案例，包括一个律师朋友怎么靠3条朋友圈当月接2个案子。名额有限，你有兴趣的话我给你留一个？

**邀约渠道：**
- 私域1v1邀约（主渠道，转化率最高）
- 公众号推文推送
- 朋友圈定向@提醒
- 合作渠道转发

### 3. 现场流程设计

| 时间 | 环节 | 内容 |
|------|------|------|
| 0:00-0:05 | 开场破冰 | 自我介绍+现场调研（"谁的朋友圈发了没人看？"） |
| 0:05-0:20 | 痛点共鸣 | "为什么高客单价业务的朋友圈特别难做" |
| 0:20-0:50 | 案例拆解 | 3个真实案例（律师/花店/咨询师） |

| 1:40-1:55 | Q&A+见证 | 现场答疑+老客户视频见证 |
| 1:55-2:00 | 促单收尾 | 限时优惠+报名引导 |

### 4. 促单环节设计

**现场专属优惠：**
- 会销专属价：¥5,800（比早鸟价再省¥1,000）

**促单话术：**
> "这个价只限今天，而且只有10个名额。不是饥饿营销，是我真的只能带这么多人。7天加速营，我每天要审每个人的朋友圈、给反馈，人多了质量保证不了。想报名的，现在扫这个码。"

**现场工具：**
- 报名二维码（大屏幕+小卡片）
- 实时报数（"已有X位报名，还剩Y个名额"）
- 已报名客户见证（现场连线或视频）
