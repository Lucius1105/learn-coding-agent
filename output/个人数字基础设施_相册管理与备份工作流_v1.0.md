## 一、文档定位（为什么存在）

### 1.1 核心问题
- iPhone 相册混乱，找不到需要的照片
- 视频素材散落在各处，剪辑时调用困难
- 证件资料急需时找不到，不急需时占空间
- 工作和生活照片混在一起，隐私风险
- 截图堆积如山，有价值的内容被淹没

### 1.2 解决目标
- **3秒找到任何照片**（按场景快速定位）
- **一键调用素材剪辑**（NAS素材库规范命名）
- **证件安全且便捷**（手机+NAS双重备份）
- **生活工作分离**（隐私保护+内容复用）
- **截图24小时处理**（有价值存档，无价值删除）

### 1.3 服务对象
- **直接用户**：老徐（徐沛霖）
- **间接受益**：雨后甘霖内容生产、阿徐文化个人品牌
- **使用场景**：日常拍摄、内容创作、证件使用、生活记录

## 二、总体架构（金字塔顶层）

```
老徐的数字生活（四大模块）
│
├── 🏢 【雨后甘霖】工作主战场（内容生产）
├── 👤 【阿徐文化】个人品牌（IP建设）
├── 🏠 【老徐生活】个人世界（生活记录）
├── 🛡️ 【证件资料】重要档案（身份凭证）
└── 🛠️ 【系统工具】辅助流程（临时中转）
```

**设计原则**：
- **MECE**：相互独立，完全穷尽，不重叠不遗漏
- **金字塔**：顶层战略（4大模块）→ 中层战术（18个相册）→ 底层执行（命名规范）
- **2-5-18法则**：2大领域（工作+生活）→ 5大模块 → 18个相册
- **3秒原则**：任何照片3秒内找到归属相册

### 4.2 素材流转规则

| 阶段 | 位置 | 操作 | 保留时间 |
|------|------|------|---------|
| **拍摄** | iPhone「最近项目」 | 拍摄完成 | 即时 |
| **导入** | NAS「01_待剪辑素材」 | AirDrop/USB | 1-7天 |
| **剪辑** | 剪映专业版（Mac） | 调用NAS素材 | 剪辑期间 |
| **归档** | NAS「02_原始素材」 | 按类型分类 | 永久 |
| **使用** | NAS「03_已使用素材」 | 标记已使用 | 永久 |
| **废弃** | NAS「04_废弃素材」 | 定期清理 | 1个月 |

### 4.3 与手机相册的关系

```
手机相册（热数据）          NAS素材库（冷数据）
├── 最近3个月照片            ├── 全部历史照片（icloudpd自动）
├── 常用素材（头像/封面）     ├── 视频素材库（手动归档）
├── 待发布内容               └── 项目归档（剪辑后移动）
└── 已发布成品

不重复存放原则：
- 小文件（<10MB）：手机保留，NAS自动备份
- 大文件（>100MB）：手机只留缩略图，原片存NAS
- 视频原始素材：剪辑后删除手机版，保留NAS版
```

## 五、第三层：完整工作流程

### 5.1 场景A：口播视频生产（完整流程）

```
【拍摄阶段】
打开「开拍」→ 应用美颜预设 → 拍摄多条口播
    ↓
素材保存到 iPhone「最近项目」
    ↓
精选片段 → 移动到「雨后甘霖/01_待发布/拍摄中」
    ↓
AirDrop 到 Mac 或直接剪辑

【剪辑阶段】
剪映专业版 → 导入素材
    ↓
剪辑完成 → 导出成品
    ↓
成品保存到「雨后甘霖/02_本周发布/周三」
    ↓
原始素材（大文件）→ 移动到 NAS「02_原始素材/授课素材」
    ↓
删除手机原始素材（释放空间）

【发布阶段】
当天发布 → 移动到「雨后甘霖/03_已发布/视频号」
    ↓
截图数据（播放量/点赞）→ 同文件夹保存
    ↓
icloudpd 夜间自动同步到 NAS
```

### 7.2 自动同步机制

```
iPhone 照片
    ↓

    ↓
icloudpd（NAS每日自动同步）
    ↓
NAS /docker/icloudpd/data/2026/03/
    ↓
（大文件手动归档）→ /video-素材库/
```

### 7.3 手动归档时机

| 触发条件 | 操作 | 目标位置 |
|---------|------|---------|

## 九、常见问题排查

### Q1：iPhone 存储空间不足
**解决**：
1. 设置 → 照片 → 开启「优化 iPhone 储存空间」
2. 删除已备份到 NAS 的大视频原始素材
3. 清理「最近删除」相册

### Q2：找不到某张照片
**解决**：
1. 检查「系统/01_待整理」（可能未分类）
2. 打开 HashPhotos → 智能搜索
3. 检查 iCloud 网页版（可能未同步）

### Q3：NAS 同步失败
**解决**：
1. 检查 icloudpd 容器状态（DSM → Container Manager）
2. 查看日志：是否有 2FA 验证过期
3. 手动触发同步：重启 icloudpd 容器

### Q4：证件急需时找不到
**解决**：
1. 固定存放「证件资料/01_个人证件」
2. 开启 Face ID 锁定，但记住位置
3. 紧急情况：NAS 网页版下载备份
