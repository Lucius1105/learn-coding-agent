# OpenClaw 产出目录

> **状态：待审核区**

此目录为 OpenClaw 产出的待审核区。所有文件需老徐审核通过后，由 CC 归档到 Obsidian 核心资料库。

---

## 目录结构

```
output/
├── README.md                          # 本文件
│
├── AI朋友圈印钞机v2.0/               # AI朋友圈印钞机项目
│   ├── _待审核_当前版本/              # 当前版本核心文档
│   ├── _待审核_多角色产出/            # 多角色迭代产出
│   ├── _待审核_朋友圈排期/            # 按日期的排期和文案
│   └── _待审核_其他/                  # 头马文案、README等
│
├── 小龙虾赋能营/                     # 小龙虾赋能营项目
│   └── _待审核/                       # 所有待审核文件
│
├── 内容产出/                         # 内容生产相关
│   ├── _待审核_视频号脚本/            # 视频号脚本
│   ├── _待审核_多平台分发/            # 多平台分发内容
│   └── _待审核_朋友圈日志/            # 朋友圈日志和台账
│
├── 知识库/                           # 知识库素材
│   ├── _待审核_每日简报/              # 每日简报
│   ├── _待审核_案例素材库/            # 案例、设备清单等
│   └── _历史存档_已归档/              # 历史归档文件
│
└── GTD系统/                          # GTD和时间管理
    └── _待审核/                       # GTD相关文档
```

---

## 文件命名规范

- 日期格式：`YYYYMMDD_`
- 版本号：`_v1.0`、`_v2.0` 等
- 示例：`2026-04-05_完整项目方案_v2.3.md`

---

## 产出规则

1. **统一放到对应项目的 `_待审核` 目录下**
2. **文件名包含日期和版本号**
3. **不要自行归档，等老徐审核**
4. **不要删除任何文件，只移动**
