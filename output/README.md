# OpenClaw 产出目录

> 此目录为 OpenClaw 产出的**待审核区**。
> 
> **所有文件需老徐审核通过后，由 CC 归档到 Obsidian 核心资料库。**
> 
> 规则：
> - 所有新产出文件统一放到对应项目的 `_待审核` 目录下
> - 文件名包含日期和版本号
> - 不要自行归档，等老徐审核
