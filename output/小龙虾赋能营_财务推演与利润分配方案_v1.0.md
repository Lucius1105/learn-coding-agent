## 二、MVP最小模型（30人单场）详细测算

### 收入
```
 gross revenue（总收入）    ¥14,940
```

### 直接成本（Cash Cost）

| 项目 | 金额 | 备注 |
|------|------|------|
| 场地费 | ¥4,000 | 4小时，含投影、音响 |
| 伴手礼+物料 | ¥1,000 | 笔记本、笔、资料打印 |
| 广告印刷摄影 | ¥750 | 取中间值 |
| **直接成本合计** | **¥5,750** | - |

### 毛利润（Gross Profit）
```
¥14,940 - ¥5,750 = ¥9,190（毛利率61.5%）
```

### 隐性成本（必须计入）

| 项目 | 估算价值 | 说明 |
|------|---------|------|
| 老徐讲课备课 | ¥3,000 | 按市场价半天课折算 |
| 现场助理1人 | ¥500 | 演示/签到/拍照 |
| 渠道分佣(20%) | ¥2,988 | 假设6人来自渠道 |
| **隐性成本合计** | **¥6,488** | - |

### 真实净利润
```
¥9,190 - ¥6,488 = ¥2,702
真实利润率：18.1%
```

**结论：MVP模型几乎不赚钱，目的是验证市场和打磨产品。**

## 六、关键风险提示

### 高风险项

| 风险 | 概率 | 影响 | 应对策略 |
|------|------|------|---------|
| **招生不达预期** | 中(40%) | 现金流断裂 | 先收定金再开课，降低固定成本 |
| **深度转化率低** | 中高(50%) | 利润大幅缩水 | 提升课程质量，加强 nurture |
| **竞品低价竞争** | 中(30%) | 被迫降价 | 差异化定位，强调实战落地 |
| **交付能力不足** | 低(20%) | 口碑崩塌 | 控制接单量，保证服务质量 |

### 财务安全线

```
生存底线：每月至少开1场（30人），否则无法覆盖固定成本
健康水平：每月2-3场，有盈余投入增长
理想状态：每月4场+，快速积累案例和口碑
```

## 七、给老徐的建议

### 立即行动

1. **先做MVP验证**（3月内完成1-2场30人课）
   - 目标：验证课程内容、收集案例、打磨话术
   - 盈亏平衡即可接受，不要追求利润

2. **设定止损线**
   - 如果前3场都招不满20人，暂停项目重新评估
   - 最大可承受损失：¥20,000（时间和金钱）

3. **利润分配预留弹性**
   - 前期（1-100人）：老徐拿50%+，团队30%，留存20%
   - 中期（100-300人）：老徐40%，团队35%，留存25%
   - 后期（300人+）：老徐35%，团队40%，留存25%

### 关键指标监控

| 指标 | 健康值 | 预警值 | 危险值 |
|------|--------|--------|--------|
| 单场报名人数 | ≥25人 | 15-24人 | <15人 |
| 深度转化率 | ≥10% | 5-9% | <5% |
| 客户满意度 | ≥90% | 80-89% | <80% |
| 月度现金流 | 正数 | 持平 | 负数连续3月 |
