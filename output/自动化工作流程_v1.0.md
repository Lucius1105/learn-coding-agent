# 市场总监自动化工作流程 v1.0

## 核心目标
从"被动执行"升级为"主动驱动成交"的 CMO + IP 操盘手

## 每日5次主动检查（已规划）

| 时间 | 任务 | 内容 | 渠道 |
|------|------|------|------|
| 06:00 | 早间简报 | 今日选题建议+热点推送 | 飞书 |
| 11:00 | 上午检查 | 内容发布提醒+数据询问 | 飞书 |
| 14:00 | 下午检查 | 热点更新+内容建议 | 飞书 |
| 19:00 | 晚间检查 | 私域互动提醒+成交导向 | 飞书 |
| 22:00 | 当日复盘 | 数据复盘+明日预告 | 飞书 |

## 自动化组件

### 1. 热点获取（Tavily）
- **脚本**：`/skills/tavily-search/scripts/daily-briefing.sh`
- **频率**：每日5次检查前自动运行
- **输出**：AI动态、商业案例、教练行业、创业者相关

### 2. 文案生成（AI）
- **输入**：实时热点 + 内容方向排期
- **输出**：3-5条朋友圈文案
- **保存**：`output/朋友圈文案_YYYY-MM-DD.md`

### 3. 定时推送（Cron）
- **状态**：待配置（系统问题）
- **替代方案**：手动/半自动推送

### 4. 数据追踪（待建设）
- 朋友圈发布数量
- 互动数据（点赞/评论）
- 私域新增人数
- 询盘数量
- 成交转化

## 工作流程图

```
06:00 自动触发
  ↓
获取实时热点（Tavily）
  ↓
生成今日选题建议
  ↓
推送飞书给老徐
  ↓
老徐确认/调整
  ↓
生成文案或直接发布

11:00 / 14:00 / 19:00 / 22:00 重复检查流程
```
