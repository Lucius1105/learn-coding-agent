## 高管智囊团任务分配

### 阶段一：CPO（产品经理-乔布斯）
**任务：** 搭建交付骨架
**时间：** 2小时

**交付物：**
1. **60分钟SOP（5幕结构）**
   - 幕0 · 预诊（会前1天）：3道预诊问题 + 老徐5分钟备课
   - 幕1 · 破冰共鸣（0-8分钟）：基于预诊信息主动共鸣
   - 幕2 · 灵魂拷问（8-30分钟）：连续追问3-5层"为什么"
   - 幕3 · 破局给方向（30-45分钟）：3个"这周就能做"的具体行动
   - 幕4 · 种升单种子（45-55分钟）：预判下一个卡点，埋下种子
   - 幕5 · 收尾留钩子（55-60分钟）：确认行动项 + ¥299抵扣说明

2. **时间成本核算**
   - 总投入时间≤70分钟（60分钟诊断 + 5分钟备课 + 5分钟审核AI报告）
   - 时薪≥¥500底线

3. **服务边界划定**
   - 不替老板做决策
   - 不代运营
   - 不做24小时响应

### 阶段二：CMO（市场总监-周受资）
**任务：** 注入灵魂包装
**时间：** 2小时

**交付物：**
1. **痛点场景描述**

   - 3个典型场景

2. **价值翻译**
   - 把SOP翻译成客户能感知的体验
   - 5个"哇"时刻设计

3. **朋友圈种草文案（3条）**
   - 说明商业急诊室是什么、为什么值¥299
   - 不卖焦虑，不low
   - 有案例感

4. **微信私聊邀约话术**
   - 有人问"你们做什么的"时怎么介绍
   - 有人表示"最近生意不好"时怎么自然引出急诊室
   - 邀约成功后的确认话术

5. **品牌一致性检查**
   - Clean Fit高智感美学
   - 客观、克制、一针见血
   - 不贩卖焦虑，不搞噱头

### 阶段三：VP Sales（销售总监-雷布斯）
**任务：** 铺设转化路径
**时间：** 2小时

**交付物：**
1. **3条分流转化路径**
   - 路径A：获客焦虑型 → 推荐AI朋友圈变现系统
   - 路径B：转化瓶颈型 → 推荐1v1销售陪跑
   - 路径C：方向迷茫型 → 推荐战略梳理服务

   每条路径包含：
   - 诊断中识别该卡点的关键信号
   - 幕4种种子的具体话术
   - 升单推荐的表达方式（不推销，对症下药）
   - 客户顾虑和化解话术

2. **跟进时间表**
   - D1：发送诊断报告
   - D3：验证预判 + 软性升单
   - D7：总结回顾 + 最后机会

3. **"预判验证"策略（3个场景）**
   - 场景1：预判获客卡点
   - 场景2：预判执行卡点
   - 场景3：预判团队卡点

4. **升单话术和异议处理**
   - ¥299全额抵扣（7天内升级）
   - ROI算账话术（帮客户算"不解决的代价"）
