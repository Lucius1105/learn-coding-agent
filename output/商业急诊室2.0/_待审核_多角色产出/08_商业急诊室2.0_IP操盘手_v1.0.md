## 一、短视频策略定位

### 1.1 平台选择与定位

#### 视频号（主战场，70%精力）

**选择理由：**
- 与朋友圈打通，私域导流最直接
- 用户画像匹配：30-45岁创业者、中小企业主
- 算法友好：社交推荐+内容质量双驱动
- 老徐已有基础：粉丝虽不多但精准

**内容定位：**
- 深度干货 + 真实案例 + 人格化表达
- 时长：1-3分钟（完播率优先）
- 风格：口播为主，场景化拍摄为辅

**核心指标：**
- 爱心率 > 3%（社交推荐触发）
- 完播率 > 30%
- 评论率 > 1%
- 私信转化率 > 2%

#### 小红书（辅助阵地，20%精力）

**选择理由：**
- 年轻创业者聚集地（25-35岁）
- 搜索流量大，长尾效应强
- 图文+视频双形态，内容复用率高

**内容定位：**
- 轻量干货 + 视觉化呈现 + 情绪共鸣
- 时长：30秒-1分钟
- 风格：快节奏、高密度、强视觉冲击

**核心指标：**
- 收藏率 > 5%（核心指标）
- 展示点击率 > 8%
- 互动率 > 3%

#### 抖音（同步测试，10%精力）

**策略：**
- 同步视频号内容，测试流量反馈
- 根据数据优化前3秒钩子
- 不单独制作，降低运营成本

### 1.2 内容调性定位

#### 老徐语言风格（必须遵守）

**要做的：**
- 短句为主，一句话一个观点
- 故事化开头，用场景代入
- 反问金句，引发思考
- 数字化感受，具体可信
- 口语词自然，像聊天

**不要做的：**
- 说教体（"你应该..."）
- AI体（"首先...其次...最后..."）
- 贩卖焦虑（"再不学就晚了"）
- 推销腔（"限时福利""错过再等一年"）
- 连续感叹号（!!!）

#### 内容DNA三原则

1. **真** — 真实经历、真实案例、真实数据
2. **善** — 利他导向，帮客户解决问题
3. **美** — 专业感、向往感、可信赖
