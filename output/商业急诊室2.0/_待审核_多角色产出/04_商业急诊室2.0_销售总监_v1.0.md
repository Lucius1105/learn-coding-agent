## 二、销售场景设计

### 2.1 线上场景

#### 场景1：朋友圈触达

**场景描述：** 通过朋友圈内容吸引意向客户主动咨询

**内容类型：**
| 类型 | 频率 | 内容方向 | 示例 |
|------|------|---------|------|
| 案例型 | 每周2条 | 客户诊断案例（脱敏） | "昨天诊断了一个做烘焙培训的学员，发现她的问题不是缺客户，而是..." |
| 洞察型 | 每周2条 | 行业痛点分析 | "为什么你的私域有5000人，却卖不动货？" |
| 价值型 | 每周1条 | 方法论/工具分享 | "3步诊断法，快速找到生意卡点" |
| 促销型 | 每月2次 | 限时优惠/名额 | "本月最后3个早鸟名额，¥199体验商业急诊室" |

**转化路径：**
```
朋友圈内容 → 点赞/评论 → 私聊互动 → 需求挖掘 → 产品介绍 → 成交
        ↓
    直接私信咨询 → 快速响应 → 产品介绍 → 成交
```

**关键动作：**
- 每条朋友圈设置「钩子」（问题/悬念/利益点）
- 评论区主动互动，引导私聊
- 私信咨询5分钟内响应

---

#### 场景2：私聊转化

**场景描述：** 客户主动私信咨询或老徐主动触达潜在客户

**触发时机：**
1. 客户在朋友圈/社群表达相关痛点
2. 老客户推荐的新客户
3. 内容互动后的跟进
4. 活动/直播后的咨询

**转化路径：**
```
首次接触 → 需求挖掘 → 价值呈现 → 异议处理 → 成交促单 → 交付跟进
   ↓           ↓           ↓           ↓           ↓           ↓
 3分钟       5分钟       5分钟       3分钟       2分钟       持续
```

**关键动作：**
| 阶段 | 动作 | 工具/话术 |
|------|------|----------|
| 首次接触 | 快速响应，建立信任 | 标准欢迎语 |
| 需求挖掘 | 提问了解痛点 | SPIN提问法 |
| 价值呈现 | 匹配产品价值 | 案例+数据 |
| 异议处理 | 化解顾虑 | LSCPA模型 |
| 成交促单 | 推动决策 | 限时优惠+名额 |
| 交付跟进 | 确保体验 | 7天跟进SOP |

---

#### 场景3：社群转化

**场景描述：** 在付费社群或老徐自己的社群中进行销售

**社群类型：**
| 类型 | 特点 | 销售方式 |
|------|------|---------|
| 老徐的免费社群 | 信任度高，但付费意愿参差 | 内容种草+限时活动 |
| 老徐的付费社群 | 精准度高，转化率高 | 专属优惠+优先预约 |
| 合作社群 | 流量大，需建立信任 | 价值分享+软植入 |

**转化路径：**
```
社群价值分享 → 建立专业信任 → 痛点共鸣 → 产品植入 → 私聊转化 → 成交
```

**关键动作：**
- 定期在社群分享诊断案例（脱敏）
- 回答群友问题时自然植入产品
- 社群专属优惠（如「本群专属早鸟价」）

---

### 2.2 线下场景

#### 场景4：活动/沙龙现场

**场景描述：** 线下活动、沙龙、分享会现场销售

**活动类型：**
| 类型 | 规模 | 销售策略 |
|------|------|---------|
| 老徐主办的沙龙 | 20-50人 | 现场专属优惠，限时报名 |
| 合作方活动 | 50-200人 | 演讲植入+现场咨询台 |
| 行业峰会 | 200+人 | 品牌曝光+精准客户跟进 |

**转化路径：**
```
活动参与 → 内容价值感知 → 现场咨询 → 扫码支付 → 即时交付问卷 → 预约诊断
```

**关键动作：**
- 演讲内容植入诊断案例
- 现场设置「急诊室体验角」（15分钟快闪诊断）
- 现场专属优惠（如「今天现场报名¥199，再送1个月答疑」）

---

#### 场景5：转介绍

**场景描述：** 老客户推荐新客户

**转介绍触发点：**
1. 急诊室客户体验满意后
2. R90客户取得阶段性成果后
3. 老客户主动询问能否推荐朋友

**转化路径：**
```
老客户满意 → 主动推荐/邀请推荐 → 新客户咨询 → 优先响应 → 专属优惠 → 成交
```

**关键动作：**
- 交付后主动请求推荐
- 推荐双方各得优惠
- 推荐客户优先预约（插队权）

---

| 转介绍 | 老客户推荐 | 50% | 1-2单 | 客户满意度+激励机制 |
| **合计** | - | - | **10-15单** | - |

## 三、定价策略

### 3.1 两档定价设计

| 价格 | 定位 | 适用场景 | 使用规则 |
|------|------|---------|---------|
| **¥199** | 早鸟体验价 | 新用户首单 | 仅限首次购买，限时48小时 |
| **¥980** | 标准价 | 正常售价 | 无限制，随时可购 |

### 3.2 早鸟价使用规则

**适用条件：**
1. 新用户（从未购买过雨后甘霖任何产品）
2. 首单（每人仅限1次）
3. 限时（咨询后48小时内支付有效）
4. 预付（支付后7天内完成诊断，过期不退）

**规则说明：**
- 设置门槛防止羊毛党
- 限时制造紧迫感
- 预付确保意向度
- 过期不退筛选高意向客户

### 3.3 老客户优惠机制

| 客户类型 | 优惠内容 | 使用规则 |
|---------|---------|---------|
| 复购客户 | ¥780（标准价8折） | 第二次购买起适用 |
| R90客户 | 免费赠送1次 | R90签约后赠送，可转赠 |
| 推荐人 | ¥50优惠券 | 成功推荐1人获得 |
| 被推荐人 | ¥50优惠券 | 首次购买可用 |
| 社群会员 | ¥780（标准价8折） | 付费社群成员专属 |

### 3.4 价格锚定策略

**价值锚定：**
```
商业急诊室价值 = ¥14,800 R90 ÷ 10 = ¥1,480
实际定价 = ¥980（标准价）/ ¥199（早鸟价）
感知价值 = 1.5倍-7.4倍
```

**对比锚定：**
- 普通商业咨询：¥2000-5000/小时
- 商业教练：¥1000-3000/小时
- 商业急诊室：¥980/45分钟（含AI诊断+报告+7天跟进）

## 四、销售话术体系

### 4.1 首次接触话术

#### 场景A：客户主动咨询

**客户：** "这个商业急诊室是什么？"

**话术：**
```
商业急诊室是45分钟的1v1深度诊断，帮你快速找到生意卡点。

流程：填问卷 → AI预诊 → 30分钟视频诊断 → 拿报告 → 7天行动跟进

交付：一份《商业诊断报告》+ 7天可执行的破局行动清单

适合：年营收100-500万、遇到增长瓶颈的创业者

价格：新用户首单¥199（原价¥980），限时48小时有效。

你现在遇到的核心问题是什么？我可以帮你判断是否适合。
```

#### 场景B：朋友圈互动后私聊

**背景：** 客户在朋友圈评论/点赞了诊断案例

**话术：**
```
看到你对刚才的案例有共鸣，你目前是不是也在遇到类似的问题？

（等客户回复后）

这个问题其实很常见，昨天刚诊断的一个客户也是卡在这里。

他之前[描述类似情况]，诊断后发现核心问题是[核心问题]，
给了[解决方案]的建议，现在[效果]。

如果你也想快速诊断一下自己的业务，可以体验一下商业急诊室。
45分钟，¥199，比你自己摸索一个月还管用。
```

### 4.2 需求挖掘话术（SPIN模型）

#### S（情境问题）：了解现状

```
- 你现在主要做什么业务？
- 做了多长时间了？
- 团队有几个人？
- 主要的获客渠道是什么？
```

#### P（难点问题）：发现痛点

```
- 现在最大的挑战是什么？
- 哪个环节最让你头疼？
- 如果只能解决一个问题，你会选哪个？
- 这个问题困扰你多久了？
```

#### I（暗示问题）：扩大痛点

```
- 这个问题如果不解决，接下来会有什么影响？
- 你算过这个问题每月让你损失多少吗？
- 团队是不是也被这个问题拖累？
- 你有没有试过什么方法？效果如何？
```

#### N（需求-效益问题）：引导价值

```
- 如果这个问题解决了，对你的业务会有什么改变？
- 你觉得值多少钱来解决这个问题？
- 如果45分钟能给你一个清晰的破局方向，你愿意试试吗？
```

### 4.3 异议处理话术（LSCPA模型）

#### 异议1："价格太贵了"

**L（倾听）：** "你觉得¥199/¥980的价格超出预算了，是吗？"

**S（分担）：** "理解，创业每一分钱都要花在刀刃上。"

**C（澄清）：** "不过你可以算笔账：现在这个问题如果不解决，
每个月损失的可能不止¥980。
而且你自己摸索，试错成本可能更高。"

**P（陈述）：** "商业急诊室45分钟，AI+人双重诊断，
带走一份可执行的方案，比你自己琢磨一个月还管用。
很多客户诊断完第一句话就是'早知道早来了'。"

**A（要求）：** "这样，你先填个问卷，AI预诊是免费的，
看看能不能帮到你，再决定要不要继续，可以吗？"

---

#### 异议2："我需要考虑一下"

**L（倾听）：** "你想再考虑一下，是对哪方面还有顾虑吗？"

**S（分担）：** "没问题，重大决策确实需要慎重。"

**C（澄清）：** "能具体说说你的顾虑是什么吗？
是担心效果，还是时间安排，还是其他？"

**P（陈述）：** "其实大多数客户考虑后还是会来，
因为问题不会因为考虑而自动解决。
而且早鸟价只有48小时，过期就恢复¥980了。"

**A（要求）：** "这样，我先给你预留一个名额，
你先填问卷做AI预诊（免费的），
看看诊断结果，再决定要不要预约，怎么样？"

---

#### 异议3："我不确定有没有用"

**L（倾听）：** "你担心诊断完还是找不到方向，是吗？"

**S（分担）：** "这种担心很正常，毕竟我们第一次接触。"

**C（澄清）：** "你之前有没有找过类似的咨询或教练服务？
效果怎么样？"

**P（陈述）：** "我们的不同在于：
1. AI预诊+人工诊断，双重保障；
2. 不是给建议，是给可执行的方案；
3. 7天行动跟进，确保落地。

而且不满意可以全额退款，你没有任何风险。"

**A（要求）：** "这样，你先看看这个案例[发案例]，
和你情况类似，诊断后的效果。
觉得靠谱再决定，好吗？"

---

#### 异议4："我时间不合适"

**L（倾听）：** "你最近时间安排比较紧张，是吗？"

**S（分担）：** "理解，创业者时间都不够用。"

**C（澄清）：** "你大概什么时候方便？
我们可以预约你方便的时间，提前3天预约就行。"

**P（陈述）：** "其实45分钟换一个清晰的破局方向，
比你花几小时自己琢磨要高效得多。
而且诊断后还有7天行动清单，每天只要20分钟。"

**A（要求）：** "这样，我们先约个你方便的时间，
如果临时有事可以提前24小时改期，怎么样？"
