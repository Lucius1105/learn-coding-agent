### 3.3 盈亏平衡点分析

#### 仅考虑急诊室产品

```
固定成本 = ¥8,448/年
单客变动成本 = ¥441.20
加权平均单价 = ¥500

盈亏平衡单量 = 固定成本 ÷ (单价 - 变动成本)
              = ¥8,448 ÷ (¥500 - ¥441.20)
              = ¥8,448 ÷ ¥58.80
              = 143.7单/年
              = 12单/月
```

**结论：** 单独做急诊室，需月均12单才能盈亏平衡。

#### 考虑R90转化（目标情景）

```
月均急诊室单量 = 10单
R90转化率 = 25%
月均R90单量 = 2.5单
R90单价 = ¥14,800
R90单客成本 = ¥2,000（估算）

月均总收入 = (10单×¥500) + (2.5单×¥14,800)
           = ¥5,000 + ¥37,000
           = ¥42,000

月均总成本 = ¥704(固定) + (10单×¥441.20) + (2.5单×¥2,000) + ¥2,300(其他)
           = ¥704 + ¥4,412 + ¥5,000 + ¥2,300
           = ¥12,416

月均净利润 = ¥42,000 - ¥12,416 = ¥29,584

盈亏平衡单量（考虑R90）：
设急诊室单量为X，R90单量为0.25X

总收入 = 500X + 14,800×0.25X = 500X + 3,700X = 4,200X
总成本 = 8,448 + 441.20X + 2,000×0.25X + 27,600 = 441.20X + 500X + 36,048 = 941.20X + 36,048

盈亏平衡点：4,200X = 941.20X + 36,048
            3,258.80X = 36,048
            X = 11.06单/月

考虑R90转化后，盈亏平衡点为月均11单急诊室（带动2.75单R90）。
```

**关键结论：**
1. 单独做急诊室，需月均12单盈亏平衡
2. 考虑R90转化后，月均11单即可盈亏平衡
3. 目标月均10单略低于盈亏平衡点，需通过提升转化率或客单价优化

### 4.2 LTV（客户生命周期价值）

#### 急诊室客户LTV

```
LTV = 首次购买价值 + 复购价值 + 转介绍价值

首次购买价值 = ¥500（加权平均）
复购价值 = ¥780 × 30%复购率 × 2次平均复购 = ¥468
转介绍价值 = ¥50优惠券 × 20%转介绍率 × 2人 = ¥20
R90转化价值 = ¥14,800 × 25%转化率 = ¥3,700

急诊室客户LTV = ¥500 + ¥468 + ¥20 + ¥3,700 = ¥4,688
```

#### R90客户LTV

```
R90客户LTV = ¥14,800（季度陪跑）
           + ¥14,800 × 40%续约率（年度陪跑R365）
           = ¥14,800 + ¥5,920
           = ¥20,720
```

### 4.3 CAC（客户获取成本）

#### 急诊室客户CAC

```
CAC = 市场投入 ÷ 新增客户数

月均市场投入 = ¥8,000
月均新增急诊室客户 = 10单

急诊室CAC = ¥8,000 ÷ 10 = ¥800/客
```

#### R90客户CAC

```
R90客户来源 = 急诊室转化
月均R90转化 = 2.5单
R90分摊获客成本 = ¥8,000 × 25%(R90收入占比) ÷ 2.5单 = ¥800/客

或直接计算：
R90 CAC = 急诊室CAC ÷ 25%转化率 = ¥800 ÷ 0.25 = ¥3,200/客
```

**CAC与LTV对比：**

| 客户类型 | CAC | LTV | LTV:CAC | 健康度 |
|---------|------|------|---------|--------|
| 急诊室客户 | ¥800 | ¥988 | 1.24:1 | ⚠️ 偏低 |
| 急诊室→R90 | ¥800 | ¥4,688 | 5.86:1 | ✅ 健康 |

1. 单独看急诊室客户，LTV:CAC=1.24:1，偏低

4. **R90转化是提升LTV:CAC的关键**
