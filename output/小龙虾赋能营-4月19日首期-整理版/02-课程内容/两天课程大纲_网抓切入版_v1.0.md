# 小龙虾AI赋能营 · 两天课程大纲（网抓切入版）

> 先用网抓震撼老板，再逐步拆解落地

---

## 课程设计理念

### 核心逻辑
```
网抓演示（震撼）→ 应用展示（兴趣）→ 方法教学（意愿）→ 落地执行（行动）
```

### 为什么先讲网抓？
1. **视觉冲击强**：现场演示抓取竞品数据、行业信息，效果立竿见影
2. **痛点精准**：老板们都想知道竞争对手在干什么、行业趋势是什么
3. **门槛适中**：比写代码简单，比纯聊天有用，正好在"够得着"的区间
4. **延展性好**：网抓是入口，后面可以接数据分析、内容生产、客户挖掘

## Day 1：网抓震撼 + 认知重塑（6小时）

### 上午（3小时）：网抓现场演示

#### 9:00-9:30 开场钩子（30分钟）
**目标：** 用网抓震撼全场，建立信任

**内容：**
```
老徐现场演示：
"大家做建材的，想知道最近柳州有哪些新开的装修公司吗？"

→ 打开AI工具
→ 输入指令："抓取柳州最近3个月新注册的装修公司名单"
→ 30秒后，屏幕上出现：公司名称、注册时间、法人代表、联系方式

"这只是开始。"
```

**金句：** "原来要雇人花一周调研的信息，现在30秒搞定。"

---

#### 9:30-10:30 网抓三大场景（60分钟）
**目标：** 展示网抓的广泛应用，勾起兴趣

**场景1：竞品监控（20分钟）**
```
演示内容：
- 抓取竞品官网的产品价格
- 抓取竞品公众号的发布频率和内容方向
- 抓取竞品在招聘网站的岗位需求（判断业务扩张方向）

老板收获："原来可以这么了解竞争对手"
```

**场景2：客户挖掘（20分钟）**
```
演示内容：
- 抓取招标网站的相关项目信息
- 抓取行业论坛的求购帖子
- 抓取社交媒体上的潜在需求

老板收获："客户线索可以这样找"
```

**场景3：行业洞察（20分钟）**
```
演示内容：
- 抓取行业报告的关键数据
- 抓取政策文件的最新变化
- 抓取上下游产业链的价格波动

老板收获："决策信息可以这样获取"
```

#### 10:30-10:45 茶歇（15分钟）

#### 10:45-12:00 网抓实操（75分钟）
**目标：** 让每个人亲手做一次网抓

**分组实操：**
```
按行业分组：
- 建材组：抓取本地装修公司信息

老徐巡回指导，确保每人成功一次
```

**成果：** 每人带走一份刚抓取的原始数据

### 下午（3小时）：认知重塑 + 方法体系

#### 14:00-14:30 网抓的延展价值（30分钟）
**目标：** 从网抓延伸到AI的全局价值

**内容：**
```
"网抓取回来的数据，只是原材料。"

展示：
1. 抓取的数据 → AI自动整理成表格
2. 整理的表格 → AI自动生成分析报告
3. 分析报告 → AI自动生成决策建议

"这才是完整的AI工作流：
网抓（获取）→ 整理（处理）→ 分析（洞察）→ 决策（行动）"
```

---

#### 14:30-15:30 AI认知重塑（60分钟）
**目标：** 建立正确的AI思维

**核心内容：**
```
1. AI不是替代你，是放大你
   - 案例：某老板用AI省了2个人工，但生意做得更好

2. AI不是万能，是用对地方
   - 哪些活适合AI干（重复、规则明确）
   - 哪些活必须人干（创意、情感、决策）

3. AI不是未来，是现在
   - 展示：竞争对手已经在用AI降本增效
   - 不算账：不用AI的隐性成本
```

---

#### 15:30-15:45 茶歇（15分钟）

---

#### 15:45-17:00 三大AI工具介绍（75分钟）
**目标：** 建立工具认知，为Day2铺垫

**工具1：内容生产工厂（25分钟）**
```
- 写文案、脚本、朋友圈
- 网抓来的竞品文案，AI改写为己用
- 现场演示：抓取5条竞品文案 → AI改写5条原创文案
```

**工具2：数据整理助手（25分钟）**
```
- 整理网抓回来的杂乱数据
- 生成图表、报告、洞察
- 现场演示：上午抓的数据 → AI一键整理成标准表格
```

**工具3：客服接待专员（25分钟）**
```
- 自动回复常见问题
- 筛选高意向客户
- 现场演示：网抓来的客户线索 → AI自动跟进话术
```

---

#### 17:00-17:30 Day1总结 + 作业布置（30分钟）
**目标：** 巩固收获，为Day2铺垫

**总结：**
```
"今天你们亲眼看到、亲手做到了：
✓ 网抓获取信息（30秒搞定一周工作量）
✓ AI延展价值（从数据到决策）
✓ 三大工具认知（明天手把手教）"
```

**作业：**
```
今晚回去用网抓抓一个你关心的信息：
- 竞品动态
- 客户线索
- 行业数据

明天带来，我们一起用AI加工它。
```

## Day 2：工具实操 + 落地执行（6小时）

### 上午（3小时）：三大工具手把手教学

#### 9:00-10:00 工具1：内容生产工厂（60分钟）
**目标：** 每人做出自己的内容工作流

**教学内容：**
```
1. Prompt工程基础（15分钟）
   - 什么是好Prompt
   - 网抓+改写的组合技巧

2. 分场景实操（30分钟）
   - 朋友圈文案：抓取竞品 → AI改写 → 一键发布
   - 视频脚本：输入主题 → AI生成 → 微调使用
   - 销售话术：客户画像 → AI定制 → 现场演练

3. 每人产出（15分钟）
   - 用网抓的数据，生成5条自己的文案
   - 老徐点评优化
```

**带走成果：** 个人内容Prompt模板3个

---

#### 10:00-10:15 茶歇（15分钟）

---

#### 10:15-11:15 工具2：数据整理助手（60分钟）
**目标：** 让杂乱数据变废为宝

**教学内容：**
```
1. 数据清洗（20分钟）
   - 网抓回来的杂乱表格 → AI一键整理
   - 格式统一、去重、补全

2. 数据分析（20分钟）
   - 自动计算关键指标
   - 生成可视化图表
   - 输出洞察结论

3. 报告生成（20分钟）
   - 数据 → 分析 → 报告，一键完成
   - 现场演示：上午抓的数据 → 下午用的报告
```

**带走成果：** 数据整理Prompt模板2个

---

#### 11:15-12:00 工具3：客服接待专员（45分钟）
**目标：** 建立自动客服体系

**教学内容：**
```
1. 常见问题梳理（15分钟）
   - 网抓客户常问的问题
   - 分类整理FAQ

2. AI回复配置（20分钟）
   - 设置自动回复规则
   - 配置关键词触发

3. 高意向客户筛选（10分钟）
   - AI识别购买信号
   - 自动标记优先级
```

**带走成果：** 客服回复Prompt模板3个 + FAQ清单
