# 视频演示内容清单 · 3人洽谈专用

> 高效展示，快速打动

---

## 一、演示目标

**核心目的：** 让准经销商/合伙人看到AI的实际效果，相信你能教他们落地

**演示时长：** 15-20分钟（每人）

**关键信息：**
- AI能做什么（解决什么痛点）
- 你能教什么（课程价值）
- 合作能赚什么（收益模式）

---

## 二、视频演示脚本（分3段）

### 第一段：开场钩子（3分钟）

**画面：** 老徐真人出镜，背景是办公室或书房

**台词：**
> "你是不是每天花4小时在重复工作上？写文案、整数据、回消息。一年就是1000+小时，相当于半年白干。
>
> 但如果用AI呢？一个月几百块成本，24小时在线，不会请假、不会离职、不会要涨薪。
>
> 今天我用15分钟，让你亲眼看看AI怎么干活。"

**配合动作：** 打开电脑，进入演示界面

---

### 第二段：三大场景实操（10分钟）

#### 场景1：AI写文案（3分钟）

**演示内容：**
```
1. 打开ChatGPT/Claude/国内AI工具
2. 输入Prompt："我是做建材批发的，帮我写5条朋友圈文案，突出品质和服务"
3. 展示AI生成的5条不同风格文案
4. 点评："原来要憋半小时，现在30秒出5条，选一条改改就能发"
```

**金句：** "这不是替代你的创意，是把你从重复劳动里解放出来。"

---

#### 场景2：AI整数据（4分钟）

**演示内容：**
```
1. 打开一个杂乱的Excel表格（提前准备好）
2. 复制粘贴到AI对话框
3. 输入指令："帮我把这个数据整理成标准格式，并计算每个客户的总消费金额"
4. 展示AI自动清洗后的表格
5. 再输入："根据消费金额给客户分层，标注高价值客户"
6. 展示分层结果
```

**金句：** "原来半天的工作量，现在5分钟搞定，而且不会出错。"

---

#### 场景3：AI做客服（3分钟）

**演示内容：**
```
1. 展示微信聊天界面（模拟客户咨询）
2. 把客户问题复制到AI
3. 输入："我是做商业教练的，客户问'你们课程和别家有什么区别'，请给出专业回复"
4. 展示AI生成的回复
5. 复制发送（或微调后发送）
```

**金句：** "常见问题的回复，AI比你写得还快还好。你只管处理复杂情况。"

---

### 第三段：课程与合作（5分钟）

**画面：** 切回老徐真人，展示课程大纲PPT

**台词：**
> "刚才你看到的就是我要教的三个工具：内容生产、数据整理、客服接待。
>
> 4月19日，4个小时，手把手带你做出自己的AI工作流。
>
> 有两种合作方式：
> - 你有时间有团队，做项目合伙人，单场能赚2000-3000
> - 你有人脉没时间，做渠道经销商，卖一张票赚125
>
> 你想了解哪种？"

## 三、电脑同步展示要点

### 需要提前准备的窗口

| 顺序 | 窗口内容 | 快捷键 |
|------|---------|--------|
| 1 | AI工具网页（ChatGPT/文心一言等） | 浏览器标签页 |
| 2 | Excel示例表格 | 桌面文件 |
| 3 | 微信聊天界面（模拟对话） | 手机投屏或模拟器 |
| 4 | 课程大纲PPT | PPT演示模式 |
| 5 | 财务测算表（可选） | Excel |

### 演示技巧

1. **提前登录**：所有账号提前登录好，避免现场输密码
2. **清理桌面**：只保留必要的窗口，其他全部关闭
3. **放大字体**：浏览器和编辑器字体调到150%，方便对方看清
4. **准备Plan B**：如果网络不好，准备录屏版本备用

## 五、常见问题预案

### Q1：这个AI工具要钱吗？
**回答：** "基础版免费，高级功能付费。但我会教你如何用免费版做出80%的效果。"

### Q2：我学不会怎么办？
**回答：** "开课后30分钟不满意，全额退款。而且我们有30天答疑，课后有问题随时问。"

### Q3：现在报名的人多吗？
**回答：** "首期限30人，目前已经有XX人报名了。你是通过XX介绍来的，我给你预留一个名额？"

### Q4：我能先听听看再决定合作吗？
**回答：** "当然可以。你可以先来参加首场，体验完整课程。如果觉得好，第二场我们再谈合作。"

## 六、促成签约话术

### 试探意向
> "刚才演示的内容，你觉得对你/你的朋友有帮助吗？"

### 假设成交
> "那这样，我先给你开通专属推广链接，你今晚就可以发朋友圈试试。哪怕先卖出1张，也是好的开始。"

### 限时优惠
> "这周签约的渠道商，首单佣金提高到30%（正常25%），算是给你的启动奖励。"

### 降低门槛
> "不用现在决定合作模式。你先当渠道商卖卖看，如果效果好，我们再聊合伙人。"
