## 三、第二段：三大场景实操（12分钟）

### 场景1：AI写文案（4分钟）

**提前打开：** ChatGPT或国内AI工具网页

**演示步骤：**

1. **展示输入**（30秒）
   ```
   在对话框输入：
   "我是做建材批发的，主营瓷砖和卫浴，
   帮我写5条朋友圈文案，突出品质好、服务周到，
   每条风格要不一样"
   ```

2. **等待生成**（30秒，可以说话填充）
   > "你看，它在思考..."

3. **展示结果**（1分钟）
   ```
   AI生成的5条文案：
   - 专业型："20年专注瓷砖卫浴，品质经得起时间考验..."
   - 故事型："昨天一个老客户说，用了我家的砖5年还跟新的一样..."
   - 促销型："本月特惠，满5000送价值300元美缝服务..."
   - 情感型："装修是大事，选材料我比你还较真..."
   - 互动型："猜猜这款进口瓷砖多少钱？猜对有奖..."
   ```

4. **点评金句**（2分钟）
   > "原来我要憋半小时，现在30秒出5条。
   > 我选一条改改就能发，剩下的存着备用。
   > 这就是效率差。"

---

### 场景2：AI整数据（5分钟）

**提前准备：** 一个杂乱的Excel表格（模拟真实业务数据）

**表格内容示例：**
| 客户名 | 电话 | 购买产品 | 金额 | 日期 |
|--------|------|---------|------|------|
| 张三 | 138xxxx | 瓷砖100平 | 5000 | 2024.1.15 |
| 李四 | 139xxxx | 卫浴一套 | 8000 | 2024.2.3 |
| （故意做得乱一点，有合并单元格、格式不统一） |

**演示步骤：**

1. **展示问题**（1分钟）
   > "你看这个表，我从销售那里收上来的，乱七八糟。
   > 以前我要花半天整理，现在让AI来。"

2. **复制粘贴到AI**（1分钟）
   ```
   输入指令：
   "帮我把这个数据整理成标准表格格式，
   并计算每个客户的消费总额，
   最后按消费金额从高到低排序"
   ```

3. **展示结果**（2分钟）
   > AI自动清洗后的表格，整齐规范
   > 自动计算了总额，排好了序

4. **进阶演示**（1分钟）
   ```
   再输入："把消费超过5000的客户标注为高价值客户，
   并生成一份客户分层分析报告"
   ```
   > 展示AI生成的分析报告

5. **金句**
   > "原来半天的工作量，现在5分钟搞定，而且不会出错。
   > 省下来的时间，我可以去谈新客户。"

---

### 场景3：AI做客服（3分钟）

**提前准备：** 手机微信界面投屏，或微信网页版

**演示步骤：**

1. **展示场景**（30秒）
   > "假设有个客户在微信上问我：
   '你们的课和网上免费的AI教程有什么区别？'"

2. **复制问题到AI**（30秒）
   ```
   输入："我是做商业教练的，开了个AI培训课。
   客户问'你们的课和网上免费教程有什么区别'，
   请给出专业且有说服力的回复"
   ```

3. **展示AI回复**（1分钟）
   ```
   AI生成的回复：
   "好问题。网上教程确实很多，但有三点区别：
   第一，我们是手把手带实操，不是光看视频；
   第二，针对老板场景定制，不是通用教程；
   第三，30天答疑服务，有问题随时问。
   你可以先听30分钟，不满意全额退款。"
   ```

4. **实际操作**（1分钟）
   > 复制AI的回复，微调后发回给客户

5. **金句**
   > "常见问题的回复，AI比我写得还快还好。
   > 我只需要处理复杂情况，效率翻倍。"

## 六、Prompt模板（复制粘贴用）

### 文案生成Prompt
```
我是做[行业]的，主营[产品]，
帮我写5条朋友圈文案，突出[卖点1]、[卖点2]，
每条风格要不一样（专业型、故事型、促销型、情感型、互动型）
```

### 数据整理Prompt
```
帮我把这个数据整理成标准表格格式，
并计算每个客户的消费总额，
按消费金额从高到低排序，
把消费超过[金额]的客户标注为高价值客户
```

```
我是做[职业]的，[业务介绍]。
客户问"[客户问题]"，
请给出专业且有说服力的回复，控制在100字以内
```
