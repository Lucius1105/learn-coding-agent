### 1.3 部署步骤（明早执行）

#### Step 1: 设备搬迁（5分钟）

**从电视旁搬到入户门旁：**
- 兮克交换机
- 群晖 DS916+

**检查：**
- [ ] 电源线长度是否够
- [ ] 网线长度是否够（需要1-2根短网线）

#### Step 2: 重新连线

**入户门位置连线：**
```
光猫 LAN口 ──网线── 华硕主路由 WAN口

华硕主路由 LAN口 ──网线── 兮克交换机 任意2.5G口

兮克交换机 任意2.5G口 ──网线── 群晖916 网口1
兮克交换机 任意2.5G口 ──网线── 群晖916 网口2 (链路聚合)
```

**书房位置：**
- 华硕 AC86U 保持原位
- **拔掉原来连接客厅的那根预埋网线**
- 只插电源，无线 Mesh 自动连接

#### Step 3: 配置检查

1. **华硕路由器管理界面**
   - 确认 Mesh 显示 "无线回程"
   - 确认两个节点都在线

2. **群晖 DSM**
   - 控制面板 → 网络 → 网络界面
   - 确认链路聚合已启用
   - 确认显示 2000 Mbps

3. **兮克交换机**
   - 检查端口指示灯
   - 确认连接群晖的端口显示 1G（群晖网口限制）

### 1.4 速度预期

| 测试场景 | 连接路径 | 预期速度 | 瓶颈 |
|---------|---------|---------|------|
| 客厅设备访问群晖 | 主路由→交换机→群晖 | **110MB/s** | 群晖千兆网口 |
| 书房 M4 访问群晖 | WiFi→Mesh回程→交换机→群晖 | **30-70MB/s** | 无线 Mesh |
| 书房 M4 访问互联网 | WiFi→Mesh→主路由 | 取决于宽带 | 无线 Mesh |

### 1.5 走线要求（当前）

**原则：** 所有网线沿墙边、踢脚线走，不可飞线

**客厅区域：**
- 光猫→主路由：沿电视柜后方走
- 主路由→交换机：沿地面踢脚线
- 交换机→群晖：沿电视柜后方

**临时方案：** 明早先测试，走线可以后续优化

#### 关于"猫棒"

**不是猫棒！** 猫棒是替换光猫的光模块，你不需要。

你需要的是：**光电转换器**（也叫光纤收发器、媒体转换器）

**推荐型号：**
- 兮克 SKS7300 系列（和你交换机同品牌，兼容性好）
- TP-LINK MC220L（性价比高，约 ¥150）
- 水星 MC201（更便宜，约 ¥100）

**华为企业级** 闲鱼 ¥120 的可能是 MA5626 等 ONU，不适合家用，别买。

#### 走线方案（隐形无感）

**路径：** 入户门 → 沿踢脚线 → 门框 → 走廊 → 书房

**固定方式：**
- 3M 透明胶（临时）
- 线槽/线卡（长期，白色与墙同色）
- 隐形光纤直径约 0.9mm，几乎看不见

**禁止：** 飞线、横跨地面、从门缝穿过

#### 与工程师沟通话术

如果你请电工/网络工程师上门：

> "我需要从客厅到书房拉一根隐形光纤，要求：
> 1. 沿踢脚线和门框走，不能飞线
> 2. 光纤要透明的，直径越细越好
> 3. 两端预留 2 米余量
> 4. 帮我做好冷接头，测试通光
> 5. 走线要横平竖直，转角用线卡固定"

**验收标准：**
- 光纤通光测试通过
- 两端光功率正常（>-20dBm）
- 速度测试达到 2.5Gbps

### 2.4 关于群晖的替代方案

你说得对，Claude Code 和 OpenClaw 可以帮你自动化群晖管理。

**但群晖 916+ 的硬件限制：**
- 无 2.5G/10G 网口
- 处理器较老
- 无法跑 Docker 大应用

**升级选项：**

| 方案 | 价格 | 优点 | 缺点 |
|------|------|------|------|
| 绿联 DX4600 Pro | ¥2000 | 2.5G网口，简单易用 | 软件生态不如群晖 |
| 极空间 Z4 Pro | ¥2500 | 影音功能强，上手快 | 开放性一般 |
| 群晖 DS923+ | ¥4000 | 软件生态最好 | 贵，万兆需另购网卡 |
| 自组 NAS | ¥1500-3000 | 性价比最高 | 需要动手能力 |

**建议：** 先用 OpenClaw 自动化现有群晖，等宽裕了再换 2.5G NAS。

## 第三部分：附录

### 3.1 无线 Mesh 速度测算标准

**测试方法：**
1. 书房 M4 连接华硕 AC86U 的 5G WiFi
2. iperf3 测试到群晖的速度
3. 或者 Blackmagic Disk Speed Test 测试 SMB 挂载

**速度对照表：**

| 信号强度 | 协商速率 | 实际传输 | 评级 |
|---------|---------|---------|------|
| -30 ~ -50 dBm | 867 Mbps+ | 50-70 MB/s | 优秀 |
| -50 ~ -65 dBm | 400-867 Mbps | 30-50 MB/s | 良好 |
| -65 ~ -75 dBm | 200-400 Mbps | 15-30 MB/s | 一般 |
| -75 dBm 以下 | <200 Mbps | <15 MB/s | 差 |

**优化建议：**
- 信号强度 <-65 dBm：把子路由往客厅方向挪
- 速度 <30 MB/s：考虑有线回程（隐形光纤）
