# 小龙虾赋能营 · 利润分配模型 v3.0（最终版）

> 正确顺序：销售佣金 → 运营成本 → 净利润分配

---

## 一、分配逻辑（三步法）

### Step 1：销售佣金（优先扣除）
```
总收入 × 20-30% = 销售人员佣金
```

### Step 2：运营成本（实际发生）
```
剩余收入 - 场地 - 物料 - 杂费 = 待分配利润
```

### Step 3：净利润分配
```
待分配利润：
├── 老徐个人：30-40%
└── 项目合伙人：60-70%
    （含助教、运营、执行团队）
```

---

## 二、100人招商大盘测算

### 基础假设

| 项目 | 数值 |
|------|------|
| 招生人数 | 100人 |
| 客单价 | ¥498 |
| 总收入 | **¥49,800** |

### 第一步：销售佣金（25%）

```
¥49,800 × 25% = ¥12,450

剩余：¥49,800 - ¥12,450 = ¥37,350
```

### 第二步：运营成本

| 项目 | 金额 | 说明 |
|------|------|------|
| 场地费 | ¥8,000 | 4场×¥2,000 |
| 物料印刷 | ¥3,000 | 资料、笔记本 |
| 摄影摄像 | ¥2,000 | 2场跟拍 |
| 餐饮茶歇 | ¥4,000 | 4场×¥1,000 |
| 杂费 | ¥2,000 | 应急备用 |
| **运营成本合计** | **¥19,000** | - |

```
剩余：¥37,350 - ¥19,000 = ¥18,350（净利润）
```

### 第三步：净利润分配

#### 方案A：保守型（老徐30%，合伙人70%）

| 分配对象 | 比例 | 金额 |
|----------|------|------|
| 老徐个人 | 30% | **¥5,505** |
| 项目合伙人池 | 70% | **¥12,845** |

**老徐总收益占比：** ¥5,505 ÷ ¥49,800 = **11.1%**

---

#### 方案B：平衡型（老徐35%，合伙人65%）

| 分配对象 | 比例 | 金额 |
|----------|------|------|
| 老徐个人 | 35% | **¥6,423** |
| 项目合伙人池 | 65% | **¥11,928** |

**老徐总收益占比：** ¥6,423 ÷ ¥49,800 = **12.9%**

---

#### 方案C：激励型（老徐40%，合伙人60%）

| 分配对象 | 比例 | 金额 |
|----------|------|------|
| 老徐个人 | 40% | **¥7,340** |
| 项目合伙人池 | 60% | **¥11,010** |

**老徐总收益占比：** ¥7,340 ÷ ¥49,800 = **14.7%**

## 六、关键发现

### 利润率对比

| 产品 | 毛利率 | 老徐净收益率 |
|------|--------|-------------|
| 录播课¥299 | 低 | ~12% |
| 训练营¥2,980 | 中 | ~15% |
| 企业版¥6,980 | 高 | ~20% |

### 核心结论

1. **客单价决定老徐收益占比**
   - ¥498低价课：老徐只能拿11-15%
   - ¥6,980高价课：老徐可以拿19-20%

2. **销售佣金是最大成本项**
   - 占总收入25%，不可忽视
   - 合伙人如果自己做销售，可省这部分

3. **运营成本相对固定**
   - 不会因为人多而线性增长
   - 规模越大，单人均摊越低

```
Phase 1：用企业版¥6,980验证价值
        → 老徐到手20%，快速回本
        
Phase 2：跑通后降价到训练营¥2,980放量
        → 老徐到手15%，规模化
        
Phase 3：沉淀录播课¥299自动卖
        → 老徐到手12%，被动收入
```
