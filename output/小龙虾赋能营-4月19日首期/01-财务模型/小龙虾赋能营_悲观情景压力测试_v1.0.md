## 五、生存策略：如何在悲观情况下活下去

### 策略A：极致降本（断臂求生）

| 措施 | 节省金额 | 代价 |
|------|---------|------|
| 砍掉固定工资 | ¥120,000 | 无全职团队，全靠老徐自己 |
| 用免费场地（合作） | ¥34,000 | 欠人情，不稳定 |
| 取消伴手礼 | ¥7,500 | 体验下降 |
| 零广告投放 | ¥40,000 | 纯靠口碑和私域 |
| **合计节省** | **¥201,500** | - |

**降本后利润：**
```
原毛利¥254,950 - 压缩后成本¥48,500 = ¥206,450
老徐可得（50%）：¥103,225
```

### 策略B：提价保利润（逆向操作）

不降价到¥398，坚持¥498，接受招生减半：

```
250人 × ¥498 = ¥124,500（培训收入）
13人 × ¥20,000 = ¥260,000（服务收入）
总收入：¥384,500

成本：
- 分佣：250×¥200 = ¥50,000
- 其他成本：¥100,000
- 固定成本：¥250,000
─────────────────────────────────
总成本：¥400,000

结果：亏损¥15,500
```

**此策略在悲观市场下也不可行。**

### 策略C：转型线上（唯一出路）

把线下课改为线上训练营：

```
定价：¥199/人（仅为线下40%）
目标：300人（更容易达成）
收入：300 × ¥199 = ¥59,700

成本：
- 平台费用：10% = ¥5,970
- 推广费用：¥10,000
- 老徐时间：¥10,000
- 助教：¥5,000
─────────────────────────────────
总成本：¥30,970

净利润：¥28,730
利润率：48.1%
```

**优势：**
- 无场地限制
- 可规模化
- 边际成本趋近于零
- 沉淀为录播课后可重复销售

## 七、给老徐的底线建议

### 财务红线

| 指标 | 红线 | 触发行动 |
|------|------|---------|
| 单场人数 | <20人 | 立即停止线下 |
| 累计亏损 | >¥10,000 | 全面转型 |
| 现金流 | 负3个月 | 砍掉所有固定成本 |
| 个人时薪 | <¥500/小时 | 不如做咨询 |

### 退出机制

如果悲观情况持续：
1. **第1个月**：尝试降价促销，测试价格敏感度
2. **第2个月**：如无效，启动线上化转型
3. **第3个月**：如仍无效，暂停项目，沉淀内容为知识付费产品

**最大可承受损失：¥20,000 + 3个月时间**
超过这个数，果断止损。

## 八、一句话总结

> **悲观情况下，线下课模型跑不通。**
> **要么极致降本（单人作战），要么转型线上（放弃重服务）。**
> **保留实力，等待市场回暖，比硬撑更明智。**
