# SKILL.md - 公众号排版官

## 技能名称
wechat-formatter - 雨后甘霖公众号排版官

## 技能描述
将文字内容转换为可直接复制粘贴到微信公众号后台的排版格式。输出为HTML文件，包含完整的样式定义，老徐只需打开文件→全选→复制→粘贴到公众号编辑器即可。

## 触发条件
用户发送 `/format` 或 `/wechat` 指令，或明确要求"排好版"、"公众号排版"、"生成HTML"等。

## 工作流程（与老徐确认版）

```
Step 1: 老徐提供内容（口述/文字/主题）
    ↓
Step 2: 我润色、扩写、确认定稿
    ↓
Step 3: 我直接排版生成HTML
    ↓
Step 4: 我返回HTML文件 + 配图建议清单
    ↓
Step 5: 老徐在M4上打开HTML预览 → 确认
    ↓
Step 6: 老徐复制粘贴到公众号后台 → 按建议配图 → 发布
```

**我的交付物：**
1. HTML文件（已排版完成）
2. 配图建议清单（标注在HTML中）
3. 视频号导流占位区
4. 相关文章链接区（需老徐填入真实链接）

**老徐负责：**
1. 内容审核确认
2. 按建议挑选/生成配图
3. 在公众号后台插入视频号卡片
4. 填入相关文章真实链接
5. 最终发布

## 雨后甘霖公众号排版规范 V2.0

### 1. 色彩系统（严格遵循VI）

| 用途 | 颜色 | HEX |
|------|------|-----|
| 主标题 | 冷静蓝 | `#0D1A63` |
| 副标题/强调 | 睿智蓝 | `#060E9F` |
| 高亮/点缀 | 丰收金 | `#FFB547` |
| 正文 | 深岩灰 | `#414042` |
| 背景 | 空气白 | `#FFFFFF` |
| 引用背景 | 浅灰蓝 | `#F5F7FA` |
| 分割线 | 淡蓝灰 | `#E8EBF0` |
| 链接 | 链接蓝 | `#1A5FBD` |

### 2. 字体规范

- **正文字号**：15px（根据人人都是产品经理文章优化）
- **标题字号**：H1=22px, H2=18px, H3=17px
- **行高**：1.75倍
- **字间距**：0.3px
- **段间距**：20px
- **两侧边距**：20px（手机端16px）

### 3. 内容结构（金字塔原理）

```
开头钩子（1-2段）→ 引发共鸣或提出痛点
    ↓
核心观点（3-5段）→ 分点阐述，每段一个观点
    ↓
案例/故事（1-2段）→ 真实案例佐证
    ↓
行动号召（1段）→ 软性引导，不直接推销
    ↓
视频号导流 → 占位区，后台手动插入卡片
    ↓
相关文章推荐 → 链接列表，放在文末
    ↓
作者简介 → 统一格式
```

### 4. 排版元素

**引用块：**
- 浅灰蓝背景 `#F5F7FA`
- 左边框3px冷静蓝
- 圆角右侧6px

**重点高亮：**
- 丰收金下划线效果
- 或冷静蓝粗体

**配图占位区：**
- 浅蓝背景 + 虚线边框
- 标注：配图编号、建议内容、风格要求
- 老徐按建议找图替换

**视频号导流占位区：**
- 金黄色渐变背景
- 提示文字："发布时在后台插入视频号卡片"

**相关文章推荐：**
- 放在文章结尾（非开头）
- 浅灰背景卡片
- 箭头+链接形式
- 建议放3-4篇相关文章

**作者简介：**
- 头像（或首字占位）
- 姓名+头衔
- 一句话介绍
- 公众号名称

### 5. 文件命名规范

`公众号_{文章标题}_{日期}_v1.0.html`

示例：`公众号_创业者最常见的三个思维陷阱_20260314_v1.0.html`

## 输出格式

生成标准HTML文件，包含：
1. 完整的CSS样式（内嵌在style标签中）
2. 符合雨后甘霖VI的所有视觉定义
3. 配图占位区（带建议说明）
4. 视频号导流占位区
5. 相关文章推荐模块
6. 作者简介模块
7. 响应式设计，适配手机阅读

文件保存至 `~/.openclaw/workspace/output/wechat/`

## 使用示例

**用户输入：**
```
/format
标题：创业者最常见的三个思维陷阱

内容：
过去13年，我陪跑过60多位创业者...

## 陷阱一：用战术勤奋掩盖战略懒惰
...

## 陷阱二：把规模当成果
...

## 陷阱三：忽视现金流
...

结尾：如果你也在创业路上遇到了卡点和困惑...
```

**我的输出：**
1. 生成HTML文件
2. 返回文件路径
3. 说明：
   - 文件已保存到 output/wechat/
   - 共3处需要配图（已标注在HTML中）
   - 相关文章链接需要填入真实URL
   - 视频号卡片需在公众号后台手动插入

## 注意事项

1. **不直接推送公众号**：仅生成排版文件，老徐需手动复制粘贴
2. **图片需单独处理**：HTML中的配图占位区，老徐需按建议找图替换
3. **链接需手动填入**：相关文章推荐区的链接为占位符，需填入真实URL
4. **视频号需后台插入**：HTML中仅提供占位区，实际卡片需在公众号后台添加
5. **预览检查**：建议老徐粘贴后在公众号预览中检查效果
