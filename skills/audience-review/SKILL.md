# 观众审稿官（点映团·小龙虾版）

> **什么时候用：** 每次内容生成后，以三个角色分别点评，找盲点。
> **目的：** 发布前找问题，避免发出去才发现。
> **参考：** 影视飓风AI点映团机制 + CC点映团v1.0

---

## 触发方式

用户发送：`/观众`
或在生成内容后自动触发。

---

## 三个审稿角色

### 角色一：卡住的创业老板（目标客户视角）

**人物设定：**
- 在柳州或广西二三线城市做生意，年营收200-500万
- 生意做了3-7年，感觉卡住了，但说不清楚卡在哪里
- 刷到这条内容，决定要不要继续看/关注

**他会问的问题：**
- 「这说的是我的问题吗？」
- 「这个人懂我吗，还是在说空话？」
- 「看完之后我能得到什么？」
- 「这个人可信吗？他经历过这些吗？」

**审稿重点：**
- 共鸣感：开头有没有说到他的状态？
- 真实感：有没有具体细节/案例，还是全是大道理？
- 可信度：有没有让他觉得「这个人经历过」？
- 行动力：看完之后他知道下一步怎么做吗？

---

### 角色二：已关注的观望者（朋友圈老读者视角）

**人物设定：**
- 已经关注老徐，加了微信，看过几条内容
- 有一定信任基础，但还没付费，在观望
- 这条内容会影响他「要不要主动来聊」

**他会问的问题：**
- 「和上次的内容有没有新东西？」
- 「看完之后我有没有更清楚找老徐的理由？」
- 「这条内容值得转发给我的朋友吗？」

**审稿重点：**
- 新鲜感：有没有比上次更深的洞见？
- 转化力：有没有让他更接近「主动来聊」这个行动？
- 传播力：他愿意转发给谁？转发的理由是什么？

---

### 角色三：算法和平台（数据视角）

**平台设定：** 根据发布平台切换——
- 朋友圈：完播率 / 爱心（转发）率 / 评论率
- 视频号：完播率 / 爱心率 / 评论率 / 转发率
- 小红书：收藏率 / 展示点击率 / 关键词密度
- 公众号：阅读完成率 / 在看 / 分享

**他会问的问题：**
- 「开头3秒/前50字，用户会不会划走？」
- 「内容节奏有没有拖沓的地方？」
- 「有没有让用户想收藏/转发的'钩子'？」
- 「关键词布局合不合理？」

**审稿重点（视频）：**
- 开头2-3秒：是否直接进入核心，没有废话
- 节奏：每3-5秒有没有新信息/新画面
- 结尾：有没有引导观众做一个动作（评论/关注/转发）

**审稿重点（图文）：**
- 前50字：是否有钩子，让人想看下去
- 段落节奏：有没有超过5行的大段落（手机看很累）
- 收藏钩子：有没有「值得保存」的信息（列表/金句/方法）

## 审稿流程

```
生成内容
    ↓
真善美自检（SOUL.md标准）
    ↓
/观众 审稿（三个角色）
    ↓
综合评估
    ↓
通过 → 给老徐确认
不通过 → 重写 → 再次审稿
```

---

```
亮点保留：...

【审核结论】
□ 通过（可直接给老徐确认）
□ 不通过（需重写）
```

## 与老徐的工作流程

1. **A1989生成内容** → 真善美自检 → /观众审稿
2. **审稿通过** → 给老徐确认
3. **审稿不通过** → A1989重写 → 再次审稿
4. **老徐确认** → 发布 → 归档到飞书多维表格

**注意：** CC/CW只负责战略升级和框架迭代，审稿由A1989独立完成。
