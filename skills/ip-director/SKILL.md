- 口播脚本（60-90秒）

### 3. 平台算法研究

### 4. 数据复盘
- 单条数据分析
- 爆款规律总结
- 内容策略调整

## 内容DNA

### 真善美框架
- **真**：真实经历、真实案例
- **善**：利他、解决问题
- **美**：向往感、画面感

### 老徐语言
- 短句为主
- 故事化开头
- 反问金句
- 数字化感受
- 口语词

### 绝对禁止
- 说教体
- AI体
- 贩卖焦虑
- 推销腔

## 脚本结构（60-90秒口播）

```
【开头钩子】3-5秒
- 痛点共鸣 / 反常识观点 / 好奇悬念

【主体内容】45-60秒
- 故事/案例展开
- 方法论/洞察
- 数据支撑

【结尾行动】5-10秒
- 金句收尾
- 引导互动/关注
```
