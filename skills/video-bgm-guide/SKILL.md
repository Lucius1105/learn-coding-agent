# 短视频配乐技能 · video-bgm-guide

## 你的任务

帮老徐为短视频选择合适的背景音乐。你要给出：
1. 风格建议
2. 具体曲库来源
3. 音量技术规范
4. 对商业教练内容的特别提醒

---

## 商业教练内容配乐原则

**核心原则：背景音乐是「空气」，不能变成「主角」**

- 听完视频，观众记住的是老徐说的话，不是背景音乐
- 音乐音量：比人声低 **20-30dB**（若人声是 0dB 基准，音乐在 -20 到 -30dB）
- 用途：填补沉默的空气感，不是烘托情绪
- 禁止：带人声的歌曲、有明显旋律高潮的流行曲

## 风格选择矩阵

| 内容类型 | 推荐风格 | 具体关键词 |
|---------|---------|----------|
| 思维框架讲解（白板类） | 轻量钢琴 / Lo-fi | lo-fi beats, soft piano, minimal |
| 观察类（在外拍摄） | 轻量吉他 / Acoustic | acoustic background, coffee shop jazz |
| 故事/案例类 | 温暖弦乐 | light strings, cinematic minimal |
| 励志/行动号召 | 轻量电子 | uplifting minimal, soft electronic |

**老徐主力内容（白板口播）：Lo-fi 钢琴 / Lounge 轻音乐**

## 音量设置步骤（剪映操作）

1. 导入视频素材（含人声）
2. 添加背景音乐到音频轨道
3. 点击音乐轨 → 调节音量
4. **目标音量：把音乐音量拖到原始音量的 10-20%**
   - 如果默认是 100%，拖到 15% 左右
   - 播放预览：说话时能隐约听到音乐即可
5. 人声轨保持 100% 不变
6. 注意：如果视频有沉默段（思考间隙），音乐会突然变明显，可以将那段音乐淡出处理
