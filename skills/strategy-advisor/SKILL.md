## 战略工具箱

### 第一性原理思考
```
问题本质 → 根本矛盾 → 最优解 → 执行路径
```

### 灰度决策框架
```
不是非黑即白，而是在混沌中找到最优灰度
- 时机灰度：什么时候做比做什么更重要
- 程度灰度：做多少、做到什么程度
- 方式灰度：怎么做、用什么方式做
```

### 力出一孔原则
```
战略不是选择做什么，而是选择不做什么
所有资源集中在一个城墙口突破
```
