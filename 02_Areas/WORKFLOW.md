## 一、收到指令时的预处理流程

### 触发条件
当收到老徐的指令时，**必须**先判断是否需要预处理：

**需要预处理的情况：**
- [ ] 指令超过3条以上文字描述
- [ ] 涉及复杂任务（多文件、多步骤）
- [ ] 目标不够清晰（做什么、给谁看、什么时候要）
- [ ] 有多个选项需要决策

**不需要预处理的情况：**
- [x] 简单查询（"查一下XX"）
- [x] 单一步骤执行（"帮我改这行字"）
- [x] 明确的是/否问题

### 预处理对话模板

```
老徐，我收到你的指令了。为了确保理解正确，我先跟你确认几点：

【目标对齐】
1. 我们要达成的最终结果是：______？
2. 这个成果的受众是：______？
3. 截止时间要求是：______？

【范围确认】
4. 包含的内容范围：______？
5. 不包含的内容：______？
6. 有特殊要求吗：______？

请确认或补充，我再开始执行。
```

### 示例

**老徐指令：**
> "帮我写一份小龙虾赋能营的宣传文案，要有海报、朋友圈、视频脚本，还要准备渠道合作的话术，然后看看用什么报名工具比较好"

**A1989预处理回复：**
> 老徐，我收到指令了。为了确保理解正确，我先跟你确认：
> 
> 【目标】为4月19日的小龙虾赋能营准备全套营销物料？
> 【受众】企业家、创业者（B端）还是AI爱好者（C端）？
> 【时间】什么时候要初稿？
> 【范围】海报指背景图还是完整设计？渠道主要指商会还是也包括个人KOL？
> 
> 请确认或补充，我再开始执行。

## 二、执行后的标准收尾流程

### 必须执行的5个步骤

#### Step 1：结果确认
向老徐展示执行结果，等待确认。

```
【执行完成】
已完成：______
产出文件：______
关键数据：______

请确认是否满足要求，需要调整的地方告诉我。
```

#### Step 2：同步到Obsidian
自动将产出文件复制到Obsidian镜像目录。

```bash
# 执行脚本
~/.openclaw/workspace/scripts/push-to-obsidian.sh
```

**同步规则：**
- 财务类 → `01-AI线下培训课-财务模型/`
- 内容类 → `02-AI线下培训课-课程内容/`
- 营销类 → `03-AI线下培训课-营销物料/`
- 执行类 → `04-AI线下培训课-活动执行/`
- 渠道类 → `05-AI线下培训课-渠道合作/`
- 存档类 → `99-AI线下培训课-存档资料/`

#### Step 3：归档整理
根据文件性质决定处理方式：

| 文件类型 | 处理方式 |
|---------|---------|
| 正式产出 | 放入对应分组文件夹 |
| 过程草稿 | 保留在output根目录或archive子目录 |
| 历史版本 | 移动到`99-存档资料/`并标注版本号 |
| 临时测试 | 删除或标记为test_前缀 |

#### Step 4：更新MOC
修改 `AI线下培训课_MOC_项目总览_v1.0.md`：
- 添加新文件链接
- 更新状态标记
- 更新最后修改时间

#### Step 5：主动通知
向老徐发送完成通知：

```
✅ 任务完成通知

【完成事项】______
【产出文件】______
【存放位置】______
【更新时间】______

是否需要推送到M4主机？
[ ] 立即推送
[ ] 稍后我自己拉取
[ ] 不需要
```

## 三、特殊情况处理

### 情况1：老徐说"紧急，马上做"
跳过预处理，直接执行，但事后补确认：
```
收到，立即执行！完成后我会跟你确认细节。
```

### 情况2：老徐给模糊指令
如"把这个优化一下"，必须先问清楚：
```
好的，请问优化的方向是：
A. 更简洁？
B. 更详细？
C. 调整语气风格？
D. 其他：______
```

### 情况3：多任务并行
如果老徐一次给多个任务，按优先级排序后逐个确认：
```
收到3个任务，我建议按以下顺序执行：
1. ______（P0，今天完成）
2. ______（P1，本周完成）
3. ______（P2，下周完成）

是否同意这个优先级？还是从哪个开始？
```

## 四、效率优化原则

### 减少往返次数
- 一次确认尽量多问几个问题
- 提供选项让老徐选择，而不是开放问答
- 预估可能的分支，提前询问

### 自动化重复动作
- 文件同步：自动执行，不询问
- MOC更新：自动执行，不询问
- 推送M4：询问后执行

### 建立默认值
对于常见场景，建立默认处理方式：

| 场景 | 默认处理 |
|------|---------|
| 文案产出 | 自动同步到03-营销物料/ |
| 财务测算 | 自动同步到01-财务模型/ |
| 课程大纲 | 自动同步到02-课程内容/ |
| 多版本文件 | 保留最新版，旧版移到99-存档/ |
